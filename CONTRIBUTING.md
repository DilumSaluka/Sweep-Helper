# Contributing

Thanks for checking out Sweep Helper! All contributions are welcome.

## Getting Started

1. **Fork** the repo
2. **Create a feature branch:** `git checkout -b feature/your-idea`
3. **Make your changes** — keep it friendly and simple
4. **Test on Windows** before submitting
5. **Push:** `git push origin feature/your-idea`
6. **Open a Pull Request**

## Guidelines

- **No ads, no telemetry, no tracking** — this app will always be clean
- **Friendly first** — every button should say what it does in plain English
- **Safety matters** — always prefer moving to restore folder over permanent delete
- **Follow existing code style** — look at how the cleaners are structured
- **Test before PR** — make sure the app still builds with `npm run build`
